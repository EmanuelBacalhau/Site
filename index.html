Principais códigos de HTML e CSS:

O que é HTML?

  HTML (Hypertext Markup Language) não é uma linguagem de programação e sim, como o nome diz, uma linguagem de marcação ou markup. Ela funciona através de “marcações” chamadas tags ou etiquetas, com o propósito de estruturar e definir a natureza do conteúdo de uma página.

Primeiro passo para começar um código HTML:

  • Digite apenas "!" em seguida pressione Enter. O código é criado automaticamente e agora só é preciso de alguns ajustes.

Níveis de títulos:

  • h1 + Enter: Nível 1;
  • h2 + Enter: Nível 2;
  • h3 + Enter: Nível 3;
  • h4 + Enter: Nível 3;
  • h4 + Enter: Nível 4;
  • h5 + Enter: Nível 5;
  • h6 + Enter: Nível 6;

Formatação de texto:

  • strong + Enter: Negrito;
  • em + Enter: Itálico;
  • u + Enter: Sublinhar texto;
  • sub + Enter: Subscrito;
  • sup + Enter: Sobrescrito;
  • p + Enter: Colocar aspas;
  • br + Enter: Quebra de linha;

Listas ordenadas e não ordenadas:

  • ol + Enter: Para criação de listas ordenadas usamos a tag incial e para identificar cada item usamos a tag li + Enter. Tipos: 1; A; a; I; i.
    Para Utilização dos tipos é necessário usar a tag type + Enter.
  • ul + Enter: Para criação de listas não ordenadas usamos a tag incial e para identificar cada item usamos a tag li + Enter. Tipos: disc; circle; square.
    Para Utilização dos tipos é necessário usar a tag type + Enter.

Carregar uma imagem HTML:

  • img + Enter: Em "src" por o local e nome da sua imagem.
    Dica: Pressione Ctrl + Espaço e o vscode vai mostrar uma lista de todos os aquivos disponíveis.

Ancorar links:

  • a + Enter: O principal atributo dessa tag é o href, que cria uma referência hipertexto ou em uma imagem. Para poder controlar onde o site de destino vai abrir, podemos usar o atributo target + Enter, que suporta os seguintes valores mais utilizados: _blank(Em uma nova janela), _self(Mesma janela ou frame atual). Imagens Flexíveis:

  • Adaptar conteúdo ao tamanho da sua tela e fazer o navegador carregar a imagem certa para cada situação. Com isso, devemos conhecer as tags picture + Enter e source + Enter.

Adicionar vídeos:

  • Para inserir um vídeo em nosso site, podemos utilizar a tag video + Enter, caso o arquivo esteja hospedado no próprio servidor.

O que é CSS?

  CSS (Cascading Style Sheets) é uma linguagem de criação de estilos, que quando combinados determinam qual será a aparência de uma página web. Enquanto o HTML cuida da estrutura contida em uma página, é o CSS quem diz ao navegador com o que cada parte da página deve se parecer.

Primeiro passo para ligar o css ao html:

  • Utilizamos a tag link:css + Enter para trabalhar com arquivos externos de estilo. Essa tag deve ser colocada dentro da área head.

Aplicando cores:

  • color + Enter: Aplica a cor nas palavras;
  • background-color + Enter: Cor da página, ou usar em algum modelo caixa;

Fontes:

  • font-size + Enter: O tamanho padrão da fonte geralmente é 16px, equivalente a 1em. Podemos configurar usando os dois tipos de tamanho, "em" sendo 2 vezes maior que a fonte padrão;
  • font-family + Enter: Aplicar fonte desejada;
  • font-style + Enter: Aplicar itálico;
  • font-weight + Enter: Aplicar negrito;
  • text-align + Enter: Alinhamento de texto;

Modelo caixa:

  • Header + Enter: Cria áreas relativas a cabeçalhos. Pode ser o cabeçalho principal de um site ou até mesmo o cabeçalho de uma seção ou artigo. Normalmente inclui títulos h1 - h6 e subtítulos. Podem também conter menus de navegação.
  • Nav + Enter: Define uma área que possui os links de navegação pel estrutura de páginas que vão compor o website. Um nav pode estar dentro de um header.
  • Main + Enter: É um agrupador usado para delimitar o conteúdo principal do nosso site. Normalmente concentra asseções, artigos e conteúdos periféricos.
  • Section + Enter: Cria seções para sua página. Ela pode conter o conteúdo diretamente no seu corpo ou dividir os conteúdos em artigos com conteúdos específicos. Segundo a documentação oficial da W3C, “uma seção é um agrupamento temático de conteúdos, tipicamente com um cabeçalho”.
  • Article: Um artigo é um elemento que vai conter um conteúdo que pode ser lido de forma independente e dizem respeito a um mesmo assunto. Podemos usar um articlepara delimitar um post de blog ou fórum, uma notícia, etc.
